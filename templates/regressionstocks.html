Stock Prices vs S&P 500

Multiple Linear Regression

Using sklearn and statsmodels libraries in Jupyter Notebook, we were able to run a multiple linear regression on the 2019 11 sectors' (individual companies) closing stock prices vs the S&P 500 Index.The OLS (Ordinary Least Squares) Regression Results image below is the output provided from the statsmodels library. The R-squared value is 0.991 which is close to 1 meaning that the multiple linear regression model is a good fit with the our data.


It is important to point out that the Telecommunication Services and Real Estate P-Values are greater than 0.05, meaning that there is no siginificant relationship between these two variables and S&P500 Index.
