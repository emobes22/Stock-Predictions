DATA


The first table contains 31 years of data between 1987-2017. S&P 500 and CPI data were downloaded from DataHub.io as a csv file. Disaster data was extracted from Emergency Events Database (EM-DAT) as a csv file. Both datasets were restricted to 1987-2017 to allow comparable datapoints for the machine learning portion of our project. The data was cleaned using Pandas in Jupyter Notebook.

  • Materials
  • Health Care

  • Industrials
